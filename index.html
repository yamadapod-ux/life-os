📐 Test Statistics
t-test unknown σ
t = (x̄ − μ₀) / (s / √n)
df = n − 1  |  SE = s / √n
z-test mean known σ หรือ n ≥ 30
z = (x̄ − μ₀) / (σ / √n)
SE = σ / √n
z-test proportion binary data
z = (p̂ − p₀) / √[p₀(1−p₀) / n]
SE = √[p₀(1−p₀) / n]
Confidence Interval
CI = x̄ ± tα/2, df × (s / √n)
ถ้า μ₀ อยู่นอก 95% CI → Reject H₀ ที่ α = 0.05
⚖️ Decision Rules
Two-Tailed
H₁: μ ≠ μ₀
Reject H₀ ถ้า
|t| > tα/2, df
Upper One-Tail
H₁: μ > μ₀
Reject H₀ ถ้า
t > tα, df
Lower One-Tail
H₁: μ < μ₀
Reject H₀ ถ้า
t < −tα, df
p-value Rule
ใช้ได้ทุกกรณี
Reject H₀ ถ้า
p < α

⚠️ Errors & Power
Type I Error
P = α
Reject H₀
ทั้งที่ H₀ จริง
(False Positive)
Type II Error
P = β
Fail to Reject
ทั้งที่ H₀ เท็จ
(False Negative)
Power of Test
Power = 1 − β
Power ↑ เมื่อ: n↑  |  α↑  |  Effect size↑  |  σ↓
α ↓ → Type I Error ↓ แต่ Type II Error ↑ (tradeoff)
n ↑ → ลดทั้ง Type II Error และเพิ่ม Power โดยไม่กระทบ α
